##### \# Atividad SpeechDown

##### 

##### 

##### Proyecto fullstack para gestión de discursos, con:

##### \- \*\*Backend\*\*: Java/Spring Boot

##### \- \*\*Frontend\*\*: React.js

##### 

##### \## ??? Requisitos

##### \- Java 17+

##### \- Node.js 16+

##### \- Maven

##### \- MySQL (opcional)

##### 

##### \## ?? Cómo ejecutar

##### 

##### \### Backend

##### ```bash

##### cd backend

##### mvn spring-boot:run

##### ```

##### 

##### \### Frontend

##### ```bash

##### cd frontend

##### npm install

##### npm start

##### ```

##### 

##### \## ?? Variables de entorno

##### Crea un `.env` en `frontend/`:

##### ```plaintext

##### REACT\_APP\_API\_URL=http://localhost:8080

##### ```

##### 

##### \## ?? Estructura del proyecto

##### ```

##### Atividad\_speechdown/

##### +-- backend/

##### +-- frontend/

##### +-- .gitignore

##### +-- README.md

##### ```

#####
